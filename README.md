# Battle City

A browser-based recreation of Namco's classic **Battle City** (1985), with modern UI polish.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move tank |
| Space | Fire |
| Esc | Pause |
| R | Restart stage (while paused) |
| Q | Quit to menu (while paused) |

## Features

- Faithful 13×13 tile gameplay with brick, steel, forest, water, and ice
- 35 stages, 20 enemies per stage, 4 enemy types, 6 power-ups
- Classic split layout: playfield + sidebar HUD
- Animated menus, stage banners, particles, pause overlay

## Tech

- TypeScript, Vite, Phaser 3
